# MediaFlowChain
A service chain connecting various LLM services for media creation

## 주요 기능
- LLM 서비스 체인을 사용하여 텍스트, 비디오, 음악 생성.
- Python 기반 프론트엔드 API 제공.
- 영화 제작을 위한 자동화된 API 제공.
